Node SDK for the Digital Ocean v2 API.

## Getting Started

Install `digital-ocean-cli` using `yarn`:

```shell
yarn global add digital-ocean-cli
```

## Usage

```shell
$ docli --help
$ docli droplet
$ docli droplet create my-droplet --token <YOUR TOKEN>
```
